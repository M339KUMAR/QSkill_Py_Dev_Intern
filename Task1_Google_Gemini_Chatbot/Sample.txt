Suppose the conversation is:
#-----------------------------
User:
Hello

Assistant:
Hi!

User:
My name is Praveen.
#-----------------------------------
The history sent to Gemini becomes:
#-----------------------------------
[
    {
        "role": "user",
        "parts": [
            {
                "text": "Hello"
            }
        ]
    },

    {
        "role": "assistant",
        "parts": [
            {
                "text": "Hi!"
            }
        ]
    },

    {
        "role": "user",
        "parts": [
            {
                "text": "My name is Praveen."
            }
        ]
    }
]

Gemini now receives the entire conversation and can answer context-aware questions.

Instead of storing only:
{
    "role":"user",
    "content":prompt
}
We'll store
{
    "role":"user",
    "content":prompt,
    "time":get_timestamp()
}
Likewise for the assistant's response:
{
    "role":"assistant",
    "content":response,
    "time":get_timestamp()
}
This allows us to display or export when each message was sent.
